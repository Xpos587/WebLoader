## О проекте

Loader - Скачивает видео и плэйлисты с ютуба в формате mp4 и mp3

## Версия Loader V1.1.1:
* Исправлены ошибки в README.md
* Пофикшен баг со скачиванием файлов
* Скачивание фалов формата mp3 снова поддерживается

## Установка

* [Python 3](https://www.python.org/)

### Зависимости

* Pytube
  ```pip install pytube```
* Moviepy
  ```pip install moviepy```
### Дев Зависимости
* Pyinstaller
  ```pip install pyinstaller```
### Установка
0. В папке Modules находятся полезные модули (файлы)
1. Установка проекта проекта через консоль:
   ```git clone https://github.com/Xpos587/Loader.git```
2. Для запуска просто запустите `Loader.exe`
## Использование
Запустите `App.py` или `Loader.exe`.
И следуйте указаниям
